# Sentiment Analysis Project

## 📌 Objective
Classify text into positive, negative, or neutral sentiment.

## 🛠 Tools Used
- Python
- Pandas
- NLTK
- Scikit-learn

## 🤖 Algorithm
Logistic Regression

## 📊 Process
- Data Cleaning
- Text Preprocessing
- Feature Extraction (CountVectorizer)
- Model Training

## 📈 Result
Model predicts sentiment with good accuracy.
